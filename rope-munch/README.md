# rope-munch
This is the frontend for the rope munch event register system

## Dev env

### Start tailwind 
```shell
npm tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

### Start dev server
```shell
npm start
```

#### Build release
```shell
npm run build
```
